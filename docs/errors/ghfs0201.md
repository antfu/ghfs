# GHFS0201: An execute is already in progress

Code: `GHFS0201`
Level: error

## What this error means

The `ghfs` server was asked to start an execute run while a previous execute is still running. Only one execute run can be in flight at a time.

## Why this happens

- A second execute was triggered from the UI or CLI before the first finished
- The previous execute is waiting on a slow API call
- The previous execute hung and its lock wasn't released

## How to fix it

Wait for the current execute to complete. The UI surfaces per-operation progress. If it appears truly stuck, stop and restart the `ghfs` process.
