# GHFS0102: Execution cancelled

Code: `GHFS0102`
Level: error

## What this error means

You chose not to proceed when `ghfs execute` asked to confirm the queued operations. No operation was sent to GitHub.

## Why this happens

- You answered "no" at the confirmation prompt
- You hit Ctrl-C / ESC during the confirm step

## How to fix it

Re-run `ghfs execute` and confirm the operations when prompted. If the preview looks wrong, edit `execute.md` / `execute.yml` (or the per-issue frontmatter) first, then run again.

## Additional context

- This is a benign "user said no" state, not a bug
