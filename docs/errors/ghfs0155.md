# GHFS0155: Action requires at least one assignee

Code: `GHFS0155`
Level: warn

## What this warning means

An `assign`, `unassign`, or `set-assignees` action was written with no usernames after the issue reference. The line is skipped.

## Why this happens

- Username list left empty
- Only whitespace or a trailing comma after `#n`

## How to fix it

Provide one or more usernames (without the leading `@` unless you're quoting):

```md
# Wrong
assign #125

# Correct
assign #125 antfu
assign #125 antfu, octocat
```

## Additional context

- Assignees must be valid GitHub usernames who can be assigned in this repository
