# GHFS0154: Action requires at least one label

Code: `GHFS0154`
Level: warn

## What this warning means

A `label`, `unlabel`, or `set-labels` action was written with no label names after the issue reference. The line is skipped.

## Why this happens

- Trailing label list was left empty
- Only whitespace or a trailing comma was left after `#n`

## How to fix it

Provide one or more comma-separated labels:

```md
# Wrong
label #125

# Correct
label #125 bug
label #125 bug, enhancement
```

## Additional context

- Label names with spaces must be quoted: `label #125 "good first issue"`
