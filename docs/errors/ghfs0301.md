# GHFS0301: Milestone not found

Code: `GHFS0301`
Level: error

## What this error means

A `set-milestone` op references a milestone that doesn't exist on the target repository. GitHub's API rejects setting an unknown milestone, so the operation is skipped.

## Why this happens

- Milestone title typo (milestone lookup is case-sensitive)
- The milestone was deleted or renamed on GitHub since the op was written
- You pointed at the wrong repository

## How to fix it

Create the milestone on GitHub first, or reference it by its numeric id:

```md
# By title (must match exactly)
set-milestone #125 "v1.0"

# By numeric id (from the milestone URL)
set-milestone #125 4
```

Milestones can be managed at `https://github.com/<owner>/<repo>/milestones`.

## Additional context

- If you meant to remove the milestone, use `clear-milestone #125` instead
