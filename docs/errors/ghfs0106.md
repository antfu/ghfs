# GHFS0106: Invalid execute file — wrong shape

Code: `GHFS0106`
Level: error

## What this error means

`.ghfs/execute.yml` parsed as YAML, but the resulting value doesn't match the expected schema. One or more entries are missing a required field or use a disallowed type.

## Why this happens

- An operation is missing `action` or `issue`
- `labels` or `assignees` is a string where a list is expected (or vice-versa)
- An entry isn't a mapping at all (e.g. a bare string was placed where an object should be)

## How to fix it

Each operation must be a mapping with at least `action` and `issue`:

```yaml
# Correct
- action: label
  issue: 125
  labels: [bug, enhancement]

# Wrong — labels must be a list
- action: label
  issue: 125
  labels: bug
```

The error message includes a `detail` string with the specific field that failed validation.
