# GHFS0200: A sync is already in progress

Code: `GHFS0200`
Level: error

## What this error means

The `ghfs` server was asked to start a sync while a previous sync is still running. Only one sync can run at a time.

## Why this happens

- You triggered sync twice in quick succession via the UI, CLI, or API
- A long-running initial sync is still going
- The previous sync hung and its lock wasn't released

## How to fix it

Wait for the current sync to finish, then retry. The UI shows progress while a sync is running.

If you suspect the previous sync is truly stuck, stop the `ghfs` process (e.g. exit `ghfs ui`) and restart it.
