# GHFS0108: Invalid execute file — rule violations

Code: `GHFS0108`
Level: error

## What this error means

`.ghfs/execute.yml` parsed and each entry is well-shaped, but one or more operations violate per-action rules (e.g. a `label` op with no labels, or `set-title` with an empty title).

## Why this happens

- Required per-action fields are missing (empty list where at least one is required, empty string where content is required)
- Conflicting fields set on the same operation
- Values out of range (e.g. non-positive issue numbers)

## How to fix it

Read the `detail` in the error message — it lists each violating operation and the rule it broke. Then edit `.ghfs/execute.yml` to satisfy the rules:

```yaml
# Wrong — no labels provided
- action: label
  issue: 125
  labels: []

# Correct
- action: label
  issue: 125
  labels: [bug]
```
