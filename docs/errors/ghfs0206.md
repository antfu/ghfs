# GHFS0206: Queue operation failed

Code: `GHFS0206`
Level: error

## What this error means

A queue-panel operation failed with a server-side detail message. This is a catch-all wrapper for errors that don't have a more specific code yet — the actual cause is in the message body.

## Why this happens

- The server rejected the change (file system error, permission denied, parse error mid-edit)
- An upstream call failed with an error the server surfaced as-is
- An unexpected condition during queue file rewrite

## How to fix it

Read the `detail` in the error message — it describes what actually went wrong. Common fixes:

- If it's a file permission error, make sure `.ghfs/` is writable by the current user
- If it's a parse error, open the referenced file and fix the syntax
- If you can't interpret the detail, copy it into an issue on [antfu/ghfs](https://github.com/antfu/ghfs/issues) so it can be promoted to a dedicated code

## Additional context

- A future release may replace specific cases of this code with more targeted codes
