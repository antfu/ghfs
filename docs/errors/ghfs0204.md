# GHFS0204: `execute.md` not found

Code: `GHFS0204`
Level: error

## What this error means

The queue panel tried to remove or edit an op that originates from `.ghfs/execute.md`, but the file doesn't exist on disk.

## Why this happens

- `.ghfs/execute.md` was deleted or renamed while the UI was open
- `ghfs sync` / `ghfs execute` has never run in this directory, so the file was never created
- The working directory was switched out from under the server

## How to fix it

Create `.ghfs/execute.md` (an empty file is fine) and reload the UI:

```bash
touch .ghfs/execute.md
```

Or run `ghfs sync` which will set up the `.ghfs` scaffold.

## Additional context

- `execute.md` is the human-friendly command list. See the [README](../../README.md#1-executemd-recommended) for syntax
