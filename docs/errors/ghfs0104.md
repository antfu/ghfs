# GHFS0104: Action requires a pull request target

Code: `GHFS0104`
Level: error

## What this error means

The operation targets an issue, but the action can only run on a pull request (for example, PR-specific actions like merging or editing a patch).

## Why this happens

- The `#123` reference points at an issue, not a PR
- An issue number was copied in place of a PR number
- A bulk operation accidentally swept in issue numbers

## How to fix it

Point the operation at a pull request instead:

```md
# Wrong — #123 is an issue
merge #123

# Correct — #456 is a PR
merge #456
```

If you meant to act on the issue, pick an action that works on issues (e.g. `close`, `comment`, `label`).

## Additional context

- Under `.ghfs/`, issues live in `issues/` and pull requests live in `pulls/`. Check which directory the number appears in to confirm the target type.
