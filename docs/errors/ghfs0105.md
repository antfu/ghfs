# GHFS0105: Failed to parse execute YAML

Code: `GHFS0105`
Level: error

## What this error means

`ghfs` could not parse `.ghfs/execute.yml` as valid YAML. The file has a syntax error so no operations can be loaded from it.

## Why this happens

- Unbalanced quotes, brackets, or braces
- Mixed tab / space indentation
- Unescaped special characters (`:`, `#`, `-`) in unquoted strings
- Truncated file from a crashed editor

## How to fix it

Open `.ghfs/execute.yml` and fix the syntax. A minimal valid file looks like:

```yaml
# .ghfs/execute.yml
- action: close
  issue: 123
- action: label
  issue: 125
  labels: [bug, enhancement]
```

Paste your file into a YAML linter (e.g. `yq`, `yamllint`) if the error message isn't enough to locate the problem.
