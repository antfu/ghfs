# GHFS0013: Invalid repository selection

Code: `GHFS0013`
Level: error

## What this error means

`ghfs` showed a list of candidate repositories (detected from your git remotes and `package.json`) and the value picked or typed does not match any of them.

## Why this happens

- A value was typed that isn't in the candidate list
- The custom entry didn't parse as `owner/name`, a GitHub URL, or a git remote URL
- A stale cached selection no longer matches the current working directory

## How to fix it

Pick one of the detected candidates from the list, or pass `--repo` with a valid value:

```bash
ghfs --repo antfu/ghfs
```
