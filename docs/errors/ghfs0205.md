# GHFS0205: Cannot edit this op source from the queue panel

Code: `GHFS0205`
Level: error

## What this error means

The queue panel tried to edit an operation whose `source` isn't something the panel can write back to (for example, `execute.yml` or a per-item markdown file).

## Why this happens

- The op was loaded from `.ghfs/execute.yml` — YAML ops must be edited in the file directly
- The op was generated from a per-item markdown frontmatter change — edit the markdown file instead
- The source is an internal, non-user-owned source

## How to fix it

Open the source file named in the error and edit it directly:

- `.ghfs/execute.yml` → edit the YAML operation
- `.ghfs/issues/**/*.md` or `.ghfs/pulls/**/*.md` → edit the frontmatter
- Then the queue will update automatically
