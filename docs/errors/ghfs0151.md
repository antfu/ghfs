# GHFS0151: Unrecognized action pattern

Code: `GHFS0151`
Level: warn

## What this warning means

A line in `execute.md` starts with a word `ghfs` doesn't recognise as an action. The line is skipped.

## Why this happens

- Typo in the action name
- A stray sentence / prose line in the middle of the operation list
- Using a new action that isn't supported yet by this version of `ghfs`

## How to fix it

Use a supported action name, or move prose lines into a comment:

```md
# Wrong — `notes` isn't an action
notes: remember to close #1

# Correct — as a comment
# remember to close #1
close #1
```

See [GHFS0103](./ghfs0103.md) for the list of supported actions and aliases.

## Additional context

- `execute.md` also accepts `#` and `//` single-line comments and `<!-- ... -->` blocks
