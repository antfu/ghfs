# GHFS0103: Unsupported action

Code: `GHFS0103`
Level: error

## What this error means

An operation in `execute.md` or `execute.yml` names an action that `ghfs` does not know how to run.

## Why this happens

- Typo in the action name (e.g. `closes` is supported but `clsoe` is not)
- The action belongs to an older or newer version of `ghfs`
- The action was invented and isn't part of the supported set

## How to fix it

Use one of the supported action names. The main set:

- `close`, `reopen`
- `set-title`, `set-body`
- `label`, `unlabel`, `set-labels`
- `assign`, `unassign`, `set-assignees`
- `set-milestone`, `clear-milestone`
- `comment`
- `close-with-comment` (aliases: `close-comment`, `comment-close`, `close-and-comment`, `comment-and-close`)

See the [README](../../README.md#execute-operations) for the current, complete list and aliases.
