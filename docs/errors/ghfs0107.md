# GHFS0107: Invalid execute file — unknown action

Code: `GHFS0107`
Level: error

## What this error means

An operation in `.ghfs/execute.yml` names an `action` that `ghfs` doesn't recognise. Validation stops before any operation runs.

## Why this happens

- Typo in the action name
- Custom action name that isn't part of the supported set
- Mixing a newer syntax with an older `ghfs` version

## How to fix it

Use a supported action. See [GHFS0103](./ghfs0103.md) for the full list and aliases. Example:

```yaml
# Wrong
- action: set_titel
  issue: 125
  title: New title

# Correct
- action: set-title
  issue: 125
  title: New title
```
