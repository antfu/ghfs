# GHFS0012: Repository selection cancelled

Code: `GHFS0012`
Level: error

## What this error means

`ghfs` detected multiple candidate repositories (for example, a git remote and a `package.json` `repository` field pointing at different slugs) and prompted you to pick one. The prompt was dismissed before a selection was made.

## Why this happens

- You hit Ctrl-C / ESC at the repository picker
- `ghfs` is running non-interactively (CI, pipeline, redirected stdin) and cannot show a picker
- Multiple sources disagree on which repo this working directory belongs to

## How to fix it

Specify the repository explicitly so no picker is needed:

```bash
# via CLI flag
ghfs --repo antfu/ghfs
```

```ts
// or in ghfs.config.ts
export default defineConfig({
  repo: 'antfu/ghfs',
})
```
