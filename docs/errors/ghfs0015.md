# GHFS0015: Could not resolve repository

Code: `GHFS0015`
Level: error

## What this error means

`ghfs` ran with no `--repo` flag, no `repo` field in `ghfs.config.ts`, and no detectable candidate in the working directory (no git remote, no `package.json` `repository` field pointing at GitHub). There is nothing to operate on.

## Why this happens

- The working directory is not a git repository
- The git repository has no `origin` remote
- The origin remote points somewhere other than GitHub
- `package.json` has no `repository` field, or the field doesn't reference GitHub

## How to fix it

Provide the repository explicitly:

```bash
ghfs --repo antfu/ghfs
```

Or set it in `ghfs.config.ts`:

```ts
export default defineConfig({
  repo: 'antfu/ghfs',
})
```

Or run `ghfs` from inside a cloned GitHub repo where `git remote get-url origin` returns a GitHub URL.
