# GHFS0152: Action did not match its expected syntax

Code: `GHFS0152`
Level: warn

## What this warning means

A line in `execute.md` uses a known action, but the arguments don't match the syntax that action expects. The line is skipped.

## Why this happens

- Missing required arguments (e.g. `set-title #125` with no title)
- Arguments in the wrong order
- Extra trailing tokens the grammar doesn't understand

## How to fix it

Follow the syntax shown in the warning's `expects:` hint. Common forms:

```md
close #123 #234
reopen #100
set-title #125 "New title"
label #125 bug, enhancement
unlabel #125 wontfix
assign #125 antfu
comment #126 "Thanks for reporting!"
close-comment #126 "Closing as done"
```
