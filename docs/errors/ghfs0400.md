# GHFS0400: Sync context was not initialized

Code: `GHFS0400`
Level: error

## What this error means

Internal: a sync helper was called before the sync context was set up. This should never happen in normal use — it indicates a bug in `ghfs`, not something wrong with your repo.

## Why this happens

- A code path invoked a sync helper outside of the `syncRepository()` entry point
- The sync setup step threw earlier but a later step ran anyway
- Custom code is importing internal sync helpers directly

## How to fix it

If you hit this as an end user:

1. Re-run `ghfs sync` from scratch
2. If it keeps happening, file an issue with the full stack trace on [antfu/ghfs](https://github.com/antfu/ghfs/issues)

If you're extending `ghfs`, make sure you go through the public `sync` entry points rather than calling internal helpers directly.
